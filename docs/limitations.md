# Limitations

- This repository represents a Tableau Public portfolio project and is primarily intended for documentation and presentation.
- Analysis quality depends on the scope and quality of the available dataset behind the published dashboard.
- Most findings are based on dashboard-level aggregations and visual patterns rather than row-level modeling.
- The dashboard supports exploratory insights and pattern recognition, not proof of causal relationships.
- Metric definitions and source transformations may be constrained by what is exposed in Tableau Public.
- Operational decisions should not rely solely on this dashboard; additional validation, controlled analysis, and business context are required.
